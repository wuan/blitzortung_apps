# About

## Lightning Data

The base lightning data is provided by the [Blitzortung.org](https://www.blitzortung.org/) project, a community-based lightning detection network.

## Software

The software for both backend and app is provided as open source and open access:

- [Android App](https://github.com/wuan/bo-android)
- [iOS App](https://github.com/wuan/bo-ios)
- [Python Module & Server](https://github.com/wuan/bo-python)

## Running Costs

The fixed cost for the two redundant backend servers is **12.90 EUR per server, totaling 25.80 EUR per month**.
