# Über

## Blitzdaten

Die Basis-Blitzdaten werden vom Projekt [Blitzortung.org](https://www.blitzortung.org/) zur Verfügung gestellt, einem gemeinschaftsbasierten Blitzortungsnetzwerk.

## Software

Die Software für Backend und App wird als Open Source und Open Access bereitgestellt:

- [Android App](https://github.com/wuan/bo-android)
- [iOS App](https://github.com/wuan/bo-ios)
- [Python-Modul & Server](https://github.com/wuan/bo-python)

## Betriebskosten

Die festen Kosten für die zwei redundanten Backend-Server betragen **12,90 EUR pro Server, insgesamt 25,80 EUR pro Monat (309,60 EUR pro Jahr)**.
